Установка freeglut:

Скопировать Header файлы из папки freeglut installation в gl для Visual Studio C:\Program Files (x86)\Windows Kits\10\Include\10.0.18362.0\um\gl
Убедиться, что freeglut.lib файл находится в каталоге с проектом, иначе возникнет ошибка при компиляции
